Ferramentas

  • Grasshopper:

    É uma ferramenta desenvolvida pela Google para ensinar programação para iniciantes de forma interativa e divertida. A progressão é feita por níveis, recomendamos a todos a fazer os “cursos” dessa plataforma para desenvolver uma base sólida em JavaScript e ter mais facilidade no desenvolvimento do jogo.

  • Google Forms:

    será uma ferramenta de grande impacto nesse curso, pois por ele que vamos ter como o base o nível de conhecimento de cada usuário que esteja acessando ao curso.

  • YouTube:

    Por meio do YouTube, se torna possível alcançar um público pelo qual esteja interessado em aprender sobre o assunto que você esteja ensinando. Essa ferramenta nesse curso é de extrema importância pois você além de aprender a criar o jogo vai estar criando um conteúdo pelo qual ajude a outras pessoas.
